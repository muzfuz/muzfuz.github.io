Right now I'm reading:

Coding the Matrix

An introductory book to vectors and matrices through Python algorithms.

I'm working through the lessons in this book partially due to my interest in games, but also because I really, really enjoy how challenging and difficult this subject matter is. Weird, I know.

Cracking the Coding Interview

Everyone keeps telling me to read this book.

I've just started to work my way through it, and it seems extremely well thought out and laid out. The problems seem quite tricky, so I'm doing some daily code Kata.

Books I've enjoyed:

Ruby on Rails Tutorial

I've worked through this book once now, and am starting my second time through it.

I enjoyed the author's approach to TDD in this book, and although there's a lot of information for a beginner, it certainly gives you enough so that you'll know Rails better than you think you do by the end.

Raspberry Pi User Guide

This book is a great introduction to the Raspberry Pi.

I love playing around with my Pi. I haven't made anything substantial with it yet, but having such direct access to a computer's internals is awesome. Plus it now comes with Ruby! For anyone interested in getting a Pi, I highly recommend this book!

Beginning Ruby

I thought this was a great primer for the Ruby language, and programming in general.

It's the first programming book I read, so some of it went over my head at the time, but it was nontheless a great way to work through some practical examples and get building with Ruby.

Apprenticeship Patterns

More of a self-help book for an aspiring developer than anything else.

It talks about how to develop and nurture your life-long pursuit of skills and knowledge as a software developer. A great read if you're interested in that sort of thing.
